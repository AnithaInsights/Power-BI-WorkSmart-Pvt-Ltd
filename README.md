# 📊 Employee Productivity Dashboard | Power BI

## Overview
This Power BI dashboard analyzes employee productivity and workplace performance. It helps identify productivity trends, high-laziness employees, and departments that need HR attention through interactive visualizations.

## Dataset
- Total Records: 200
- Total Columns: 15

## Features
- KPI Cards
- Bar Chart
- Stacked Column Chart
- Pie Chart
- Scatter Plot
- Top 10 Lazy Employees Table
- Slicers
- Drillthrough Page
- Decomposition Tree
- What-if Parameter

## Key Insights
- Identified departments with the highest laziness.
- Compared WFH and Office productivity.
- Analyzed the impact of social media usage on productivity.
- Highlighted top-performing and high-laziness employees.

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel
